Display the numbers of MailChimp subscribers you have on your WordPress site.

== Description ==

This plugin allows you to display the number of MailChimp subscribers you have, anywhere on your WordPress site.

== Installation ==

1. Upload the plugin zip file to the `/wp-content/plugins/` directory, or search in WordPress for "MailChimp Subscriber Chiclet for WordPress"
2. Activate the plugin.
1. Go to `Settings > MailChimp Subscriber Chiclet for WordPress` and set your MailChimp API key, then copy/paste a shortcode from there to anywhere (Pages, Posts, Text Widget) in your site!

== Frequently Asked Questions ==

= Do I need a Pro MailChimp account? =

No - any level MailChimp account should work.

= How do I get my MailChimp API Key to use in the "Settings" panel? =

Go here and login: https://admin.mailchimp.com/account/api-key-popup  ... BAM!

= What colors are available for the Chiclet? =

Blue, Red, Yellow, and Black. More coming soon with the Pro version of this plugin.

= How do I add the Chiclet programmatically? =

To add the plugin to your theme (like the footer.php file), use the `do_shortcode` method of WordPress. See the screenshots.
